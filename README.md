# Tonatuna: A Dojo Gamejame Project

Tonatuna is a fully on-chain PvP fishing game.

## Running Locally

#### Terminal one
```bash
# Build contract
sozo build

# Test
sozo test
```

#### Terminal two
```bash
# Run client
cd client
pnpm install
pnpm dev
```
